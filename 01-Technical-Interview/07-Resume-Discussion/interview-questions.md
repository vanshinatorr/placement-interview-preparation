# Internship Master Story

I completed a 3-month remote internship as a Full Stack Development Intern at Plasmid Innovation.

During my internship, I worked on an existing Human Resource Management System (HRMS). Since it was an existing project, my role was to work on assigned modules instead of building the entire application from scratch.

On the frontend, I used React.js to build and update pages such as the Employee List, Employee Profile, and Leave Management. I also fixed UI issues and integrated frontend pages with backend APIs.

On the backend, I worked with Node.js and Express.js to create and modify REST APIs. Most of my work involved CRUD operations like adding, updating, deleting, and fetching employee data.

For the database, I used MongoDB with Mongoose to store and manage employee information, leave records, and user details.

I tested APIs using Postman before connecting them to the frontend. I also used Git and GitHub for version control while working on assigned tasks.

One of the main things I learned during the internship was how a full-stack application is structured. I understood how the frontend communicates with the backend through REST APIs and how data is stored and retrieved from the database.

The internship also improved my debugging skills. Whenever I faced issues such as API errors, database connection problems, or frontend bugs, I learned how to identify the root cause, fix the issue, and test the solution before completing the task.

Overall, this internship gave me practical exposure to React.js, Node.js, Express.js, MongoDB, REST APIs, Git, GitHub, and Postman, and helped me understand how developers collaborate on a real-world web application.
